# Vigilância - Notificações de Violações

Sistema municipal de notificações de violência e violação de direitos, com foco em:
- registro intersetorial por órgãos da rede de garantia de direitos;
- consolidação de dados em base única municipal;
- apoio à geração de relatórios e dashboards;
- suporte futuro à exportação para CEMARIS e SINAN.

## Stack
- Frontend: Vue 3 + Vite + PrimeVue
- Backend: Django + DRF
- Banco de dados: PostgreSQL
- Infraestrutura de desenvolvimento: Docker

## Estrutura
- `frontend/`: interface do usuário
- `backend/`: API e regra de negócio
- `docs/`: documentação funcional e arquitetural
- `docker/`: arquivos de infraestrutura
